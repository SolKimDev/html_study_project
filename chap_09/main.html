  1. CSS 포지셔닝과 주요 속성들

    CSS 포지셔닝이란 브라우저 화면 안에 각 콘텐츠 영역을 어떻게 배치할지를 결정하는 것으로, float속성과 position 속성 등을 이용하며, 여기에 박스 모델의 padding이나 border 속성 등을 포함해 전체적인 레이아웃을 결정합니다.
    이번 장에서는 CSS의 포지셔닝을 이용해 웹 문서의 레이아웃을 구성하는 방법을 알아봅니다.

    • box-sizing 속성 : 박스 너비 기준 정하기

      CSS의 width 속성의 경우 콘텐츠 영역의 너비를 나타내기 때문에 해당 요소에 적용한 패딩이나 테두리 크기는 따로 계산해서 배치해야 합니다.
      이럴 때 box-sizing 속성을 사용하면 콘텐츠 영역의 너비에 패딩과 크기까지 합쳐서 width 속성을 지정할 수 있습니다.
      기본형은 다음과 같습니다.

      box-sizing: content-box | border-box

      박스 모델의 마진은 width 속성에 포함되지 않습니다.

      각 속성에 대한 설명은 다음과 같습니다.

      • content-box : 기본값으로, width 속성 값을 콘텐츠 영역 너비 값으로 사용합니다.
      • border-box : width 속성 값을 콘텐츠 영역에 테두리까지 포함한 박스 모델 전체 너비 값으로 사용합니다.
        따라서, 박스 모델의 너비 값(width) = 콘텐츠 영역 + 좌우 패딩 + 좌우 테두리가 됩니다.

    • float 속성 : 왼쪽이나 오른쪽으로 배치하기

      float 속성은 주로 나란히 놓을 수 없는 블록 요소를 왼쪽 구석이나 오른쪽 구석에 나란히 배치할 때 사용하는데, float 속성을 사용하면 너비 값은 콘텐츠를 표시할 때 필요한 만큼만 차지하고, 다른 요소가 들어올 만큼의 공간을 비워 둡니다.
      또한, float 속성을 이용하면 그 다음에 넣는 요소들에도 똑같은 속성이 전달되며, float속성이 더이상 필요하지 않을 경우 이를 해제해 주어야만 합니다.
      float 속성을 사용함에 있어서 유의할 점은, float 속성은 요소의 위에 떠 있는 속성이기 때문에, float를 적용한 부모 속성의 너비가 기본적으로 0으로 인식됩니다.

      float 속성을 사용하면 그 주변을 다른 요소가 감쌉니다. 따라서 이미지와 텍스트를 나란히 표시하려 할 때는 이미지에 float 속성을 사용합니다.
      이미지를 배치한 후에는 적당한 간격을 유지하기 위해 margin 속성도 함께 사용하는 것이 권장됩니다.

      float에서 사용 가능한 속성은 left, right, none 속성으로, 기본형은 다음과 같습니다.

      float: left | right | none

    • clear 속성 : float 속성 해제하기

      float 속성이 더 이상 유효하지 않을 경우, 이 속성 다음부터의 float 속성은 해제됩니다.
      float: left를 이용해 왼쪽으로 배치했다면 float:left로 종료하고, float: right를 이용했다면 clear:right를 이용해 무효화시킵니다.
      float 속성 값이 left인지 right인지 상관없이 무조건 기본 사태로 되돌리고 싶다면 clear: both를 이용하면 됩니다.
      기본형은 다음과 같습니다.

      clear: none | left | right | both

      ※clear 심화※

      float를 이용 후, 이를 무효화하지 않고 div 등의 요소를 놓을 경우, float가 영역을 차지하지 않는다고 여기고 해당 영역과 겹치기 때문에, 이를 해소할 방법이 필요합니다.
      이 문제를 해결하기 위한 방법들이 어떠한 것들이 있는지 다양한 방법들을 알아봅시다.

      1. float로 대응하기

        앞의 요소가 float인 상태에서 뒤따라오는 요소가 float가 아닐 경우 이런 문제가 발생하기 때문에, 그 다음 요소 역시 float 상태로 바꾸어 주면 해결됩니다.
        다만 이는 그 뒤의 요소들에게도 동일한 문제가 발생하기 때문에 완전한 해결책이라고 볼 수 없습니다.

      2. 부모에게 overflow:hidden 사용하기

        자식요소가 float인 부모에게 overflow:hidden을 주면 부모가 자식을 담아냅니다. 다만, 이 방법은 내용이 넘치면 안 보이기 때문에 한계가 있습니다.
        예시는 다음과 같습니다.

        ⟨style⟩
        div.wrap{overflow: hidden}
        div.box1, div.box2 {float: left}
        ⟨style⟩
        ⟨div class="wrap"⟩
        ⟨div class="box1"⟩⟨/div⟩
        ⟨div class="box1"⟩⟨/div⟩
        ⟨/div⟩

      3. 빈 요소에 clear:both 넣기

        float된 요소 밑에 빈 임의의 요소를 만들어 clear:both를 줍니다. clear는 취소하다는 뜻으로 float를 취소시킵니다.
        이건 마크업의 흐름을 깨기 때문에 권장하는 방법은 아닙니다.
        예시는 다음과 같습니다.

        ⟨style⟩
        div.box1, div.box2{float: left} br.clear{clear:both}
        ⟨style⟩
        ⟨div class="box1"⟩⟨/div⟩
        ⟨div class="box1"⟩⟨/div⟩
        ⟨br class="clear"⟩

      4. 가상요소 ::after 사용하기

        부모요소에게 ::after를 사용하며 가상요소를 만들어 줍니다. 그리고 가상요소에게 display:block 요소를 적용해 한 줄 가득차게 합니다.
        그리고 빈 콘텐츠를 넣어주고, clear를 적용시켜줍니다. 부모 뒤에 보이지 않는 가상요소를 만들어 뒤에 요소가 위로 올라가지 않게 해줍니다.
        가장 많이 사용하는 방법으로, 예시는 다음과 같습니다.

        ⟨style⟩
        div.clearfixed::after{display:block;content:"";clear:both}
        div.box1, div.box2{float: left}
        ⟨style⟩
        ⟨div class="clearfixed"⟩
        ⟨div class="box1"⟩⟨/div⟩
        ⟨div class="box1"⟩⟨/div⟩
        ⟨/div⟩

    • position 속성 : 배치 방법 지정하기

      position 속성은 웹 문서 안의 요소들을 자유자재로 배치해 주는 속성으로 HTML과 CSS를 이용해 웹 문서를 만들 때 중요하게 사용하는 속성 중 하나입니다.
      position 속성을 이용하면 텍스트나 이미지를 나란히 배치할 수 있고 여러 개의 요소를 가로나 세로로 원하는 위치에 배치할 수도 있습니다. 이렇게 요소를 다양하게 배치하려면 position 속성에서 사용하는 속성 값의 특성을 이해하고 필요에 따라 적절한 속성 값을 선택해야 합니다.
      position 속성의 기본형은 다음과 같습니다.

      position: static | relative | absolute | fixed

      각 속성에 대한 설명은 다음과 같습니다.

      • static : 기본값으로, 요소를 문서의 흐름에 맞추어 배치합니다.
      • relative : 이전 요소에 자연스럽게 여결해 배치하되, 위치를 지정할 수 있습니다.
      • absolute : 원하는 위치를 지정해 배치합니다.
      • fixed : 지정한 위치에 고정해 배치합니다. 화면에서 요소가 잘릴 수도 있습니다.

      position 속성 중 static을 제외한 나머지 속성 값에서는 좌표를 이용해 각 요소의 위치를 조절할 수 있습니다.
      이 때, 위치는 top과 bottom, left, right로 지정합니다.
      top이란 위쪽에서 얼마나 떨어져 있는지, bottom은 아래에서 얼마나 떨어져 있는지를 나타냅니다.
      좌푯값은 양수와 음수 모두 사용할 수 있습니다.
