  1. 변수와 상수

    변수형 데이터의 종류로는 String, Number, Boolean, Null 데이터 타입이 있으며, 변수의 선언 방법은 다음과 같습니다.

    let 변수명; or
    let 변수명=값;

    변수의 선언은 let 키워드를 사용하며, var 키워드는 사용 가능하나 모던 웹 표준에서는 더 이상 사용하지 않습니다.
    변수가 한 블록 내에서 이미 선언된 경우, 똑같은 이름으로 선언할 수 없습니다. 또한 선언 후 값을 할당하지 않은 경우, 변수는 Null 상태이고, 변수 값은 undifined입니다.

    상수를 선언할 경우에는 const 키워드를 이용해 다음과 같이 선언합니다.

    const a = 1;

    상수 또는 변수에서 사용 가능한 데이터 타입은 다음과 같습니다.

    • 문자형 데이터

      문자나 숫자를 큰따옴표(" ") 혹은 작은따옴표(' ')로 감싸서 사용합니다.
      작은 따옴표와 큰 따옴표 사용에있어 큰 차이는 없습니다. 기본형은 다음과 같습니다.

      let 변수명 = "사용할 문자/숫자";

      다음과 같이 문자형 데이터에 (X)HTML 태그를 포함할 경우 출력문을 적용하면 태그로 인식되어 적용됩니다.

      let tag="⟨h1⟩ String ⟨/h1⟩";

    • 숫자형 데이터

      숫자형 데이터는 바로 값을 대입해 사용 가능합니다.

      let 변수명 = 숫자;

    • 논리형 데이터

      Boolean 타입은 다음과 같이 사용 가능합니다.

      let 변수명= true or flase; or
      let 변수명= Boolean(데이터);

    • null과 undefined 데이터

      undefined는 변수에 아무 값도 등록되어 있지 않은 것을 뜻하며, null은 변수에 데이터로 null이 저장되어 있는 경우를 가리킵니다.

    • typeof

      typeof는 다음과 같이 사용하여, 지정한 데이터 또는 변수의 데이터형을 얻어옵니다.

      typeof 변수/데이터;

  2. 연산자

    자바스크립트에서 사용 가능한 연산자는 다음과 같습니다.
    비교 연산자에서 ==와 ===, !=와 !==의 차이는 숫자형과 문자형의 일치 여부까지 검사할 것인지에 대한 조건입니다.

    • 기본 연산자

      • 산술 연산자 : +, -, *, /, %
      • 대입 연산자 : =, +=, *=, /=, %=
      • 증감 연산자 : ++, --
      • 비교 연산자 : >, <, >=, <=, !=, !==, ==, ===
      • 논리 연산자 : ||, &&, !
      • 문자 결합 연산자 : (피연산자를 문자열로) +
      • 삼항 연산자 : 조건식1 ? 실행문1(true) : 실행문2(false);

    • 연산자 우선순위

      연산자간 우선순위는 다음과 같습니다. 논리 연산자 사이에서는 AND 연산을 먼저 수행합니다.

      1. ( )
      2. 단항 연산자( --, ++, ! )
      3. 산술 연산자( *, /, %, +, - )
      4. 비교 연산자( >, >=, <, <=, ==, ===, !=, !== )
      5. 논리 연산자( &&, || )
      6. 대입(복대입) 연산자( =, +=, -=, *=, /=, %= )

  3. 함수
    • 기본 함수 정의문

      자바스크립트에서 함수를 선언하는 방법은 여러 가지가 있습니다.

      • function 키워드를 이용하여 다음과 같이 선언할 수 있습니다.

        fucntion 함수명(){
        . . .
        }

      • 참조 변수를 활용해 다음과 같이 익명 함수를 선언하여 사용할 수도 있습니다.

        참조 변수 = function() {
        . . .
        }

      • 화살표 함수문법을 사용하여 function 키워드를 대체할 수 있습니다.

        참조 변수 = ( 파라미터, ) => {
        . . .
        }

    • 내장 함수

      js에 내장된 기본 내장 함수는 다음과 같습니다.

      종류설명사용 예
      parseInt()문자형 데이터를 정수형 데이터로 바꿉니다.parseInt("5.12") → 5
      parseFloat()문자형 데이터를 실수형 데이터로 바꿉니다.parseFloat("5.12") → 5.12
      String()문자형 데이터로 바꿉니다.String(5) → "5"
      Number()숫자형 데이터로 바꿉니다.Number("5") → 5
      Bolean()논리형 데이터로 바꿉니다.Boolean(5) → true
      isNaN()[?]데이터에 숫자가 아닌 문자를 포함하면 true를 반환합니다.isNaN("5-3") → true
      isNaN("53") → false
      eval()문자형 데이터를 큰따옴표가 없는 스크립트 코드로 처리합니다.eval("15+5") → 20

  4. 배열
    • 배열의 선언과 조회

      자바스크립트에서는 배열 객체를 생성할 수 있으며, 이를 만드는 방법은 다음과 같이 다양합니다.
      js에는 변수 생성 시 특별히 자료형을 명시하지 않는 것과 마찬가지로, 하나의 배열에 다양한 자료형이 담길 수 있습니다. 기본적으로 Zero-Base-Indexing이 적용되며, 객체에 대한 자세한 설명은 다음 챕터에서 확인할 수 있습니다.

      1. let 참조 변수=new Array();
        참조 변수[0]=값1; 참조 변수[1]=값2; ... 참조 변수[n-1]=값n;
      2. let 참조 변수=new Array(값1, 값2, ... 값n);
      3. let 참조 변수=new Array(값1, 값2, ... 값n);

      마찬가지로, 객체형 배열 또한 선언할 수 있습니다.

      let | var | const   배열 이름 = [ { 속성 이름 : 속성 값 ... }, ... ];

      배열을 선언한 뒤, n번째 항목을 조회하고 싶을 때는 다음 구문을 이용하여 조회할 수 있습니다.

      objects[n];

    • 배열의 내장함수

      자바스크립트의 배열은 가변 길이의 객체로, 배열에 배열에 새로운 항목을 추가하거나 제거할 수 있습니다.
      배열의 데이터에 대한 조작은 배열의 내장 함수를 이용합니다.

      배열의 내장 함수들은 다음과 같습니다.

      • 배열의 크기 알아내기

        배열의 길이는 배열의 length 값을 조회하여 알아낼 수 있습니다.

        변수명 = 배열 이름.length;

      • forEach

        배열에 대하여 for문을 대체할 수 있습니다.
        이와 같이 함수형태의 파라미터를 전달하는 것을 콜백함수라고 합니다.

        배열 이름.forEach(변수명 =⟨ {
        (이 스코프 내에서 위의 변수명을 이용할 시 for문과 마찬가지로 첫 번째 인덱스 항목부터 마지막 인덱스까지 차례로 조회합니다.)
        });

      • 배열의 원소를 변환하여 새로운 배열로 추출하기

        map은 배열 안의 각 원소를 변환 할 때 사용되며, 이 과정에서 새로운 배열이 만들어집니다.
        이와 같이 변화를 주는 함수를 파라미터로 전달하는 것을 변화함수라고 합니다.
        map 함수를 이용하여 기존 배열의 값을 제곱한 새로운 배열을 만들고자 한다면 다음과 같이 이용할 수 있습니다.

        1

        const array = [1, 2, 3, 4, 5, 6, 7, 8];
         
        const square = n => n * n;
        const squared = array.map(square);
        console.log(squared);

        square 함수를 생성하는 대신, 다음과 같이 익명의 콜백 함수를 이용해 한 줄로 처리할 수도 있습니다.
        1

      • 배열에서 원하는 값의 인덱스 알아내기

        indexOf는 원하는 항목이 몇 번째 원소인지 찾아주는 함수입니다.
        indexOf를 통해 조회할 수 있는 자료형은 숫자, 문자, 불리언입니다. 반환값은 해당 값의 인덱스입니다.

        변수명 = 배열 이름.indexOf(조회할 데이터);

      • 배열에서 일치하는 객체/배열의 인덱스 알아내기

        findIndex는 배열, 객체 등 indexOf를 통해 조회할 수 없는 항목을 조회합니다. 환값은 해당 값의 인덱스입니다.

        변수명 = 배열 이름.findIndex(변수명 =⟩ 변수명.속성 이름 === 조회할 데이터);

      • 배열에서 일치하는 객체/배열 추출하기

        find함수는 findIndex와 똑같이 사용합니다. 그러나 인덱스 대신 해당 값 전체를 반환합니다.

      • 배열에서 특정 항목만 추출하기

        filter함수는 배열에서 특정 조건을 만족하는 값들만 따로 추출하여 새로운 배열을 만듭니다.
        예를 들어, 특정 배열에서 특정 값이 false인 항목들만 추출하거나, 특정 값이 n 이상한 항목들을 추출할 수 있습니다.

        변수명 = 배열 이름.filter(변수명 =⟩ 변수명.속성 이름 조건식);

      • 배열에서 특정 항목 제거하기

        splice를 이용하면 배열에서 특정 값을 제거할 수 있습니다.

        배열 이름(기준 인덱스, 기준 인덱스로부터 지울 배열의 수);

      • 배열에서 특정 구간만 추출해 새로운 배열 만들기

        slice를 이용하면 배열에서 특정 구간을 추출한 뒤, 새로운 배열을 만듭니다.
        이 과정에서 기존의 배열에 있는 값은 변화하지 않습니다.
        사용법은 splice와 완전히 동일합니다.

      • 배열의 원소 추출하기

        shift를 이용하면 첫번째 원소를 배열에서 추출해줍니다. 추출하는 과정에서 해당 원소는 사라집니다.

        변수명 = 배열 이름.shift();

        또, pop을 이용하면 배열의 마지막 원소를 배열에서 추출합니다. 마찬가지로 해당 원소는 사라집니다.

        변수명 = 배열 이름.pop();

      • 배열의 뒤에 새 항목 추가하기

        배열의 뒷쪽에 새로운 항목을 추가할 때에는 push 함수를 사용합니다.

        배열 이름.push(값);

      • 배열의 앞에 새 항목 추가하기

        unshift는 shift의 반대로, 배열의 맨 앞에 새 원소를 추가합니다.

        배열 이름.unshift(값);

      • 배열 합치기

        배열의 뒤에 매개변수로 전달된 배열을 추가해 두 배열을 합쳐 새로운 배열을 반환합니다.

        변수명 = 배열1.concat(배열2);

      • 배열의 값을 문자열로 만들기

        join을 이용하면 배열 안의 값을 문자열 형태로 합쳐줍니다.
        join은 다음과 같이 사용할 수 있습니다.

        2
        3

        console.log(array.join()); // 1,2,3,4,5
        console.log(array.join(' ')); // 1 2 3 4 5
        console.log(array.join(', ')); // 1, 2, 3, 4, 5

      • reduce 함수

        배열.reduce((누적값, 현잿값, 인덱스, 요소) =⟩ { return 결과 }, 초깃값);
        초기값이 없는 경우, 자동으로 초깃값이 0번째 인덱스의 값이 됩니다.

        reduce는 map과 짝이며(Map/Reduce의 그 Reduce입니다.), sort, every, some, find, findIndex, includes등을 모두 구현할 수 있는 아주 강력한 부모격 함수입니다.
        reduce에 대한 이해가 쉽지 않으므로, 활용 예시를 들어 설명하고자 합니다.
        reduce를 이용해 주어진 배열에 대하여 총 합을 구하고자 한다면, 다음과 같이 이용할 수 있습니다.

        1
        2
        3
        4
        5
        6
        const numbers = [1, 2, 3, 4];
        let sum = numbers.reduce((accumulator, current, i) => {
          console.log(accumulator, current, i);
          return accumulator + current;
        }, 0);
        console.log();

        		cs

        해당 수식에 대한 출력값은 다음과 같습니다.

        >> 0 1 0 ( acc:0, curr:1, index:0 )
        >> 1 2 1 ( acc:1, curr:2, index:1 )
        >> 3 3 2 ( acc:3, curr:3, index:2 )
        >> 6 4 3 ( acc:6, curr:4, index:3 )

        출력값을 보면, 우선 acc(누적값)이 초깃값인 0부터 시작해, return하는대로 누적되는 것을 볼 수 있습니다.
        유사 함수 reduceRight를 이용하면, 요소 순회를 오른쪽부터 왼쪽으로 진행합니다.
