  1. 챕터 6-1. 글꼴 관련 스타일 ▽

    글자에서 바꿀 수 있는 글꼴 속성을 알아보고 선택할 수 있는 값을 살펴봅니다.

    • font-family 속성 : 글꼴 지정하기

      font-family 속성은 ⟨p⟩ 태그나 ⟨hn⟩태그처럼 텍스트를 사용하는 요소들에서 사용하면 적절합니다.

      기본형은 다음과 같습니다.

      font-family:⟨글꼴 이름⟩ [ ,⟨글꼴 이름⟩, ⟨글꼴 이름⟩, ... ]

      글꼴을 여러 개를 적용하는 이유는 사용자의 PC에 설정한 글꼴이 없을 경우, 차례로 다음 글꼴을 적용할 수 있도록 하기 위함입니다.
      그러나 사용자의 PC에 단 하나의 폰트도 존재하지 않을 가능성을 배제할 수는 없습니다.
      이런 경우에는 다음의 '웹 폰트'를 이용하면 해결할 수 있습니다.

    • @font-face 속성 : 웹 폰트 사용하기

      웹 폰트란 웹 문서를 작성할 때 웹 문서 안에 글꼴 정보도 함께 저장했다가 사용자가 웹 문서에 접속하면 글꼴을 사용자 시스템으로 다운로드시키는 방식입니다.
      원칙적으로는 서버에 웹 폰트를 업로드해야 하나, 구글 웹 폰트(https://fonts.google.com)등을 이용하면 손쉽게 웹 폰트를 적용시킬 수 있습니다.

      웹 폰트 적용하기

      웹 폰트를 적용하는 방법은 다음과 같습니다.

      @font-face {
      font-family : 글꼴 이름 ;
      src: local(글꼴 이름) | url(글꼴 파일 경로) [, ... ] format(파일 유형);
      }

      font-family를 통해 글꼴의 이름을 지정하면, 이후 CSS에서 글꼴 관련 속성을 조정할 때 지정한 글꼴의 이름으로 이용이 가능합니다.
      또한 src를 여럿 지정할 경우, 가장 처음에 나온 폰트가 적용 가능할 경우 (파일이 없거나 브라우저에서 지원하지 않는 확장자 등의 문제가 없는 경우)에는 해당 폰트를 적용하고, 적용이 불가능하다면 순차적으로 다음 글꼴을 적용하려 시도합니다.
      글꼴의 경우 ttf 파일은 용양이 너무 큰 관계로, eot(Embedded Open Type)이나 woff(Web Open Font Format)와 같이 웹에 적합한 파일들을 주로 사용합니다.
      웹 환경에서 사용 가능한 폰트는 woff, ttf, otf, eot, svg, svgz등이 있으나, 글꼴의 경우 글꼴을 구매했더라도 웹 폰트 사용과 같이 이용 분야에 있어 사용권 범위가 까다롭기 때문에, 반드시 사전에 확인이 필요합니다.

    • font-size 속성 : 글자 크기 조절하기

      글자의 크기는 픽셀이나 포인트를 비롯한 여러 단위로 고정값을 지정하거나, 백분율을 이용한 유동 값을 지정할 수도 있습니다. 기본형은 다음과 같습니다.

      font-size: ⟨절대 크기⟩ | ⟨상대 크기⟩ | ⟨크기⟩ | ⟨백분율⟩

      font-size에서 사용 가능한 속성은 다음과 같습니다.

      • 절대 크기 : 브라우저에서 지정한 글자 크기입니다.
        사용할 수 있는 값은 xx-small | x-small | small | medium | large | x-large | xx-large입니다.
      • 상대 크기 : 부모 요소의 글자 크기(font-size)를 기준으로 더 크게 표시하거나 더 작게 표시합니다.
        사용할 수 있는 값은 larger | smaller 입니다.
      • 크기 : 브라우저와 상관없이 글자 크기를 직접 지정합니다.
      • 백분율 : 부모 요소의 글자 크기를 기준을 해당하는 %를 계산에 표시합니다.

      font-size 중 '크기'에서 사용 가능한 단위는 다음과 같습니다.

      • em : 해당 글꼴의 대문자 M의 너비를 기준으로 크기를 조절합니다.
      • ex : x-height. 해당 글꼴의 소문자 x의 높이를 기준으로 크기를 조절합니다. 크기를 지정하지 않을 경우 body 요소의 크기 16px이 기본 값 1em으로 지정됩니다.
      • px : 픽셀. 모니터에 따라 상대적 크기가 되며, 크기를 지정하지 않을 경우 문단의 크기는 16px가 됩니다.
        px단위를 사용할 경우 폰트 크기가 고정되기 때문에 모바일 기기에서의 접속을 고려한다면 px보단 em 단위의 이용이 권장됩니다.
      • pt : 포인트. 일반 문서에서 많이 사용하는 단위입니다.

    • font-weight 속성 : 글자 굵기 지정하기

      font-wieght 속성은 글자 굵기를 조절하며, 기본형은 다음과 같습니다.

      font-weight: normal | bold | bolder | lighter | 100 | 200 | ... | 900

      사용하는 속성 값은 다음과 같습니다.

      • normal : 기본값입니다.
      • 100에서 900 사이의 수치 : 400은 normal, 700은 bold에 해당합니다.

    • font-variant 속성 : 작은 대문자로 표시하기

      font-variant의 small-caps 속성을 이용하면, 대문자를 소문자 크기에 맞추어 표시할 수 있습니다.
      기본형은 다음과 같습니다.

      font-variant: normal | small-caps

    • font-style 속성 : 글자 스타일 지정하기

      font-style 속성은 글자의 이탤릭체 표시 여부를 결정할 수 있습니다.
      기본형은 다음과 같습니다.

      font-style : normal | italic | oblique

      각 속성에 대한 설명은 다음과 같습니다.

      • normal : 기본값입니다.
      • italic : 미리 디자인된 이탤릭체로 표시합니다.
      • oblique : 단순히 글꼴을 기울여 이탤릭체로 표시합니다.

    • font 속성 : 글꼴 속성을 한꺼번에 묶어 표현하기

      font-style, font-variant, font-weight 등 글꼴 관련 속성을 한꺼번에 묶어 약식으로 표기할 수 있게 합니다.
      또한, 특정 키워드를 입력해 caption, incon, menu 등 특정 주제에 어울리는 글꼴 스타일로 표시할 수도 있습니다.
      기본형은 다음과 같습니다.

      font : ⟨font-style⟩ ⟨font-varient⟩ ⟨font-weight⟩ ⟨font-size/line-height⟩ ⟨font-family⟩
      | caption | icon | menu | message-box | small-caption | status-bar

      각 속성에 대한 설명은 다음과 같습니다.

      • font-* : font-로 시작하는 글꼴 관련 속성을 한꺼번에 나열합니다. 폰트 관련 속성은 사용하는 각 값들이 다르므로 한꺼번에 나열해 사용할 수 있습니다.
      • caption : 캡션에 어울리는 글꼴 스타일로 표시합니다.
      • icon : 아이콘에 어울리는 글꼴 스타일로 표시합니다.
      • menu : 드롭다운 메뉴에 어울리는 글꼴 스타일로 표시합니다.
      • message-box : 대화상자에 어울리는 글꼴 스타일로 표시합니다.
      • small-caption : 작은 캡션에 어울리는 글꼴 스타일로 표시합니다.
      • status-bar : 상태 표시줄에 어울리는 글꼴 스타일로 표시합니다.

      이 중, font-size와 line-height는 '12px/24px' 처럼 슬래시(/)로 연결해 함께 표현협니다.
      line-height는 줄 간격을 조절하는 속성이지만, 글자 크기와 줄 간격은 밀접한 관련이 있으므로 font-size/line-height와 같이 하나의 속성처럼 사용하기도 합니다.
      line-height에 대한 자세한 설명은 이곳에서 확인할 수 있습니다.

      앞의 내용을 바탕으로 다음과 같이 텍스트 문단의 글꼴 속성을 지정할 수 있습니다.

      p {font:16px/25px "맑은 고딕" }
      p {font:bold italic 12pt }
      p {font:120%/120% bold }


      여기서 주의할 것은, font:120%/120%로 지정했을 때 글자 크기 120%는 부모 요소의 글자 크기를 기준으로 120%만큼 표시하고 뒤에 있는 줄 간격 120%는 현재 요소의 글자 크기를 기준으로 한다는 것입니다.

  2. 챕터 6-2. 텍스트 스타일 ▽

    • color 속성 : 글자 색 지정하기

      color 속성을 사용해 글자 색을 바꿀 수 있으며, 16진수나 RGB/RGBA, HSL, 또는 색상명을 이용해 색상을 표기할 수 있습니다.
      기본형은 다음과 같습니다.

      color : 색상

    • text-decoration 속성 : 텍스트에 줄 표시하기/없애기

      text-decoration을 이용해 텍스트에 줄을 표시할 수 있습니다. 기본형은 다음과 같습니다.

      ⟨text-decoration : none | underline | overline | line-through

      속성 값에 대한 설명은 다음과 같습니다.

      • none : 기본값입니다.
      • underline : 밑줄을 표시합니다.
      • overline : 영역 위로 선을 그립니다.
      • line-through : 취소선을 그립니다.

    • text-transfomr 속성 : 텍스트 대/소문자 변환하기

      text-transfomr 속성은 텍스트를 대/소문자 또는 전각 문자로 변환합니다. 이 속성은 한글에는 영향을 미치지 않고 영문자에만 적용됩니다.
      기본형은 다음과 같습니다.

      text-transform : none | capitalize | uppercase | lowercase | full-width

    • text-shadow 속성 : 텍스트에 그림자 효과 추가하기

      텍스트에 그림자 효과를 입힙니다. 속성으로 none값이나 그림자 값을 사용할 수 있는데, 그림자 값은 쉼표로 구분해 여러 값을 지정할 수 있습니다.
      기본형은 다음과 같습니다.

      text-shadow : none | ⟨x⟩ ⟨y⟩ ⟨b⟩ ⟨color⟩

      shadow color에 rgba 값을 지정할 경우, 구버전 브라우저에서는 제대로 표시하지 못할 수도 있습니다.

    • white-space 속성 : 공백 처리하기

      텍스트와 함께 입력된 공백의 처리 방법에 대해 지정합니다. 기본형은 다음과 같습니다.

      white-space : normal | nowrap | pre | pre-line | pre-wrap

      각 속성에 대한 설명은 다음과 같습니다.

      • normal : 기본 값. 여러 개의 공백을 하나로 표시합니다.
      • nowrap : 여러 개의 공백을 하나로 표시하고, 영역 너비를 넘어가는 내용은 줄을 바꾸지 않고 계속 한 줄로 표시합니다.
      • pre : 여러 개의 공백을 그대로 표시하고, 영역 너비를 넘어가는 내용은 줄을 바꾸지 않고 계속 한 줄로 표시합니다.
      • pre-line : 여러 개의 공백을 하나로 표시하고, 영역 너비를 넘어가는 내용은 자동으로 줄을 바꿔 표시합니다.
      • pre-wrap : 여러 개의 공백을 그대로 표시하고, 영역 너비를 넘어가는 내용은 자동으로 줄을 바꿔표시합니다.

    • letter-spacing과 word-spacing 속성 : 텍스트 간격 조절하기

      letter-spacing 속성은 자간을, word-spaceing은 단어 사이 간격을 조절합니다. 기본형은 다음과 같습니다.

      letter-spacing : normal | ⟨크기⟩
      word-spaceing : normal | ⟨크기⟩

  3. 챕터 6-3. 문단 스타일 ▽

    • direction 속성 : 글자 쓰기 방향 지정하기

      아랍어 등을 위한 속성으로, 텍스트의 쓰기 방향을 지정합니다. 기본형은 다음과 같습니다.

      direction : ltr | rtl

      *(ltr : left to right, rtl : right to left)

    • text-align 속성 : 텍스트 정렬하기

      문단의 텍스트 정렬 방법을 지정합니다. 기본형은 다음과 같습니다.

      text-align : start | end | left | right | center | justify | match-parent

      각 속성에 대한 설명은 다음과 같습니다.

      • start : 현재 텍스트 줄의 시작 위치에 맞추어 ltr 언어라면 왼쪽으로, rtl언어라면 오른쪽으로 맞추어 정렬합니다.
      • end : 현재 텍스트 줄의 끝 위치에 맞추어 ltr 언어라면 오른쪽으로, rtl 언어라면 왼쪽으로 맞추어 정렬합니다.
      • left : 문단을 왼쪽 정렬합니다.
      • right : 문단을 오른쪽 정렬합니다.
      • center : 문단을 가운데 정렬합니다.
      • justify : 문단을 양쪽 정렬합니다. text-justify 속성을 이용해 세부 사항을 조정할 수 있습니다.
      • match-parent : 부모 요소를 따라 문단을 정렬합니다.
        다만 부모 요소의 속성이 start나 end일 경우, 부모 요소가 ltr인지 rtl인지에 따라 left나 right 값으로 계산해 적용합니다.

    • text-justify 속성 : 정렬 시 공백 조절하기

      text-align 속성이 justify일 경우, 글자와 단어 사이 간격의 정렬 기준을 설정합니다. 기본형은 다음과 같습니다.

      text-justify : auto | none | inter-word | distribute

      각 속성에 대한 설명은 다음과 같습니다.

      • auto : 웹 브라우저에서 자동으로 지정합니다.
      • none : 정렬하지 않습니다.
      • inter-word : 단어 사이의 공백을 조절해 정렬합니다.
      • distribute : 인접한 글자 사이의 공백을 똑같이 맞추어 정렬합니다.

    • text-indent 속성 : 텍스트 들여 쓰기

      문단의 첫 글자를 얼마나 들여 쓸지를 지정합니다. 기본값은 다음과 같습니다.

      text-indent : ⟨크기⟩ | ⟨백분율⟩

      각 속성에 대한 설명은 다음과 같습니다.

      • 크기 : 단위와 함께 들여 쓸 크기를 지정합니다. 음수 값도 사용할 수 있습니다.
      • 백분율 : 부모 요소의 너비를 기준으로 상대적 크기를 지정합니다.

    • line-height 속성 : 줄 간격 조절하기

      줄 간격은 정확한 단위와 함께 크기 값을 직접 지정하거나 글자 크기를 기준으로 숫자, 실제 크기, 백분율로 표시하며, 이 때 숫자는 줄 간격 값이 아니라 n배수를 뜻합니다.
      기본형은 다음과 같습니다.

      line-height : normal | ⟨숫자⟩ | ⟨크기⟩ | ⟨백분율⟩ | inherit

    • text-overflow 속성 : 넘치는 텍스트 표기하기

      white-space 속성을 이용해 줄 바꿈을 하지 않는 경우, 넘치는 텍스트를 어떻게 처리할지 지정합니다.
      text-overflow 속성은 해당 요소에서 overflow 속성 값이 hidden이거나 scroll, auto이면서 white-space: nowrap 속성을 함께 사용했을 경우에만 적용됩니다.
      기본형은 다음과 같습니다.

      text-overflow: clip | ellipsis

      각 속성에 대한 설명은 다음과 같습니다.

      • clip : 넘치는 텍스트를 자릅니다.
      • ellipsis : 말줄임표로 잘린 텍스트가 있다고 표시합니다.

  4. 챕터 6-4. 목록 스타일 ▽

    • list-style-type 속성 : 목록의 불릿과 번호 스타일 지정하기

      ⟨ul⟩의 경우 불릿 스타일을, ⟨ol⟩의 경우 번호 스타일을 지정할 수 있습니다.
      기본형은 다음과 같습니다.

      list-style-type : none | ⟨ul:불릿⟩ | ⟨ol:번호⟩

      ul의 불릿 스타일은 다음과 같습니다.

      • disc : 채운 원. 기본값
      • circle : 빈 원.
      • square : 채운 사각형
      • none : 불릿 없애기

      ol의 번호 스타일은 다음과 같습니다.

      1. decimal : 1로 시작하는 십진수. 기본값
      2. decimal-landing-zero : 앞에 0이 붙는 십진수.
      3. lower-roman : 소문자 로마 숫자.
      4. upper-roman : 대문자 로마 숫자.
      5. lower-alpha / lower-latin : 소문자 알파벳.
      6. upper-alpha / upper-latin : 대문자 알파벳.
      7. armenian : 아르메니아 숫자.
      8. georgian : 조지 왕조시대의 숫자.

    • list-style-image 속성 : 불릿 대신 이미지 넣기

      불릿을 원하는 이미지로 바꿀 수 있습니다. 기본형은 다음과 같습니다.

      list-style-image : ⟨이미지⟩ | none
      ⟨이미지⟩ = url(이미지 파일 경로)

      각 속성 값에 대한 설명은 다음과 같습니다.

      • none : 이미지를 사용하지 않고 list-style-type 속성에서 지정한 형태로 표시합니다. 기본값.
      • ⟨이미지⟩ : url("images.jpg")처럼 url() 키워드 안에 이미지 파일 경로를 지정합니다.

    • list-style-position : 목록에 들여 쓰는 효과 내기

      기본값은 outside이며, 기본형은 다음과 같습니다.

      list-style-position : inside | outside;

    • list-stytle 속성 : 목록 속성 한꺼번에 표시하기

      font 속성과 비슷하게, list-style 속성으로 list-style-type과 list-style-position, list-style-image 속성을 한꺼번에 표시할 수 있습니다.
      이는 다음과 같이 표기할 수 있습니다.

      ol { list-style : lower-alpha, inside; } or
      ol { list-style : none; }
